# HTML & CSS — Projetos e Práticas

Este repositório reúne os estudos, exercícios e projetos que fiz enquanto aprendia os fundamentos de **HTML e CSS**.

Aqui você encontrará exemplos práticos de marcação semântica, estilização de páginas e layout responsivo básico.

---

## 🧠 Objetivo

Meu foco ao criar este repositório foi:

- Consolidar os fundamentos de HTML5 e CSS3
- Praticar estrutura semântica e boas práticas de código
- Criar layouts simples e funcionais
- Entender como organizar conteúdos estáticos

Esses projetos fazem parte do meu processo de aprendizado e construção de base para estudos posteriores em frontend e backend.

---

## 🧪 O que você pode encontrar aqui

- **HTML semântico** com tags apropriadas
- **Estilização com CSS**
- Uso de **Flexbox** para layout
- Organização de arquivos de estilo
- Páginas estáticas que podem ser abertas diretamente no navegador
